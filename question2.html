QUESTION 2 - FORMATTING TAGS OF HTML

TAGS DEFINITION
<b> tag this tag defines bold text without attaching any special importance to it.
<i> tag this tag defines text in alternate voice or mood, and the text is displayed in italics.
<u> tag underline tag defines an underlined text within the tag.
<em> tag this tag defines emphasised text, and the contents is also displayed in italics.
<del> tag the del tag defined text that has been deleted from the document. The browser usually strike a line through deleted text.
<ins> tag defines a text that has been inserted into a document. It is displayed as underlined text also.
<big> tag this tag defines bigger text.
<sub> tag this tag defines a text or character as a subscript.
<sup> tag this tag defines a text or character as a superscript.
<mark> tag this defines marked or highlighted text.
<small> tag defines smaller text.
<strong> tag this tag defines text with strong importance, and the text is also displayed in bold.
